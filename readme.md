# Spent.io - Simple Expense Manager

Spent.io is a user-friendly expense management website built using the MERN stack (MongoDB, Express, React, Node.js) and Tailwind CSS. It helps you keep track of your daily money flow, search past transactions, manage peer debt (Udhar), and calculate loan details easily.

---

## ✨ Features

* **Easy Transaction Entry:** Add your daily **Expenses**, **Income**, or **Account Transfers** with matching forms that change automatically.
* **Smart History Search:** Search your transaction history by typing any letter. It matches words regardless of upper or lower case.
* **Udhar (Debt) Tracker:** A dedicated page to track who owes you money ("Lent") and who you owe money to ("Borrow") without cluttering your main expense list.
* **Financial Calculators:** Built-in tabs to instantly calculate **Loan EMIs** and **Simple/Compound Interest** returns.
* **Mobile-Friendly Menu:** A clean layout with a smooth sidebar drawer that slides out perfectly on phones.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, React Router, Tailwind CSS, Lucide Icons, React Toastify.
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB with Mongoose ORM.
